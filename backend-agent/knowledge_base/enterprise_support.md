# CloudDesk 企业客户专属支持

## 1. 概述

本文档定义了 Enterprise 套餐客户专属支持服务的范围、流程和保障。Enterprise 客户享有专属客户经理、优先工单通道和定制 SLA 等高级服务。

## 2. 专属客户经理

### 2.1 角色定义

专属客户经理（Customer Success Manager, CSM）是 Enterprise 客户的主要对接人，负责：

- 客户关系管理和满意度维护
- 定期业务回顾和战略讨论
- 问题升级协调和资源调配
- 产品培训和最佳实践分享
- 续约谈判和增值服务推介

### 2.2 服务标准

| 指标 | 标准 |
|------|------|
| 响应时间 | 工作时间 1 小时内回复 |
| 可用时间 | 周一至周五 09:00-18:00（客户时区） |
| 紧急联系方式 | 专属手机号 + 企业 IM |
| 月度回顾会议 | 每月 1 次 |
| 季度业务回顾 | 每季度 1 次 |
| 年度战略回顾 | 每年 1 次 |

### 2.3 CSM 分配规则

- 年合同金额 >= $50,000：高级 CSM（5 年以上经验）
- 年合同金额 $10,000-$49,999：标准 CSM（3 年以上经验）
- 年合同金额 < $10,000：共享 CSM（1 个 CSM 服务 3-5 个客户）

### 2.4 CSM 变更流程

- CSM 变更需提前 2 周通知客户
- 新旧 CSM 做好交接（客户背景、历史问题、偏好等）
- 客户可对 CSM 提出变更请求
- 变更后 1 周内完成首次会面

## 3. 优先工单通道

### 3.1 优先级保障

Enterprise 客户的所有工单自动获得以下保障：

| 优先级 | 响应时间 | 解决时间 | 目标达成率 |
|--------|---------|---------|----------|
| P0（紧急） | 30 分钟 | 1 小时 | 99% |
| P1（高） | 1 小时 | 4 小时 | 95% |
| P2（中） | 4 小时 | 8 小时 | 90% |
| P3（低） | 8 小时 | 24 小时 | 85% |

### 3.2 专属工单标签

Enterprise 客户工单自动附加 `enterprise` 标签，确保：

- 工单在队列中优先排序
- CSM 收到工单创建通知
- 超时预警阈值降低 50%
- 升级路径直达 CSM 和客服总监

### 3.3 P0 工单特殊处理

当 Enterprise 客户创建 P0 工单时：

1. 自动通知 CSM（手机 SMS + 邮件）
2. 自动通知技术负责人
3. 成立专项响应小组
4. 每 30 分钟更新一次进展
5. 工单解决后 24 小时内提交故障报告

### 3.4 工单创建偏好

Enterprise 客户可通过以下方式创建工单：

- CloudDesk 控制台（推荐）
- 专属 API 接口
- 企业 IM 集成（Slack、Teams）
- 邮件发送至专属邮箱
- 电话联系 CSM

## 4. 定制 SLA

### 4.1 SLA 定制选项

Enterprise 客户可在以下范围内定制 SLA：

| SLA 项 | 标准范围 | 定制范围 |
|--------|---------|---------|
| 月度可用性 | 99.9% | 99.9%-99.99% |
| P0 响应时间 | 30 分钟 | 15-30 分钟 |
| P0 解决时间 | 1 小时 | 30 分钟-2 小时 |
| 工单并发处理数 | 不限 | 不限 |
| 专属技术支持 | 工作时间 | 7×24（需额外付费） |

### 4.2 SLA 保障措施

为确保定制 SLA 达成，CloudDesk 提供以下保障：

- 专属技术团队待命（P0 场景）
- 自动化监控和告警
- 每月 SLA 报告并对账
- SLA 违约补偿条款写入合同

### 4.3 SLA 违约补偿

Enterprise SLA 违约补偿以合同约定为准，默认补偿条款：

| SLA 违约程度 | 补偿比例 |
|------------|---------|
| 可用性低于承诺但 >= 99.5% | 月费的 5% |
| 可用性 99.0%-99.5% | 月费的 10% |
| 可用性 < 99.0% | 月费的 25% |
| P0 响应超时 | 月费的 2%/次 |

## 5. 定制服务

### 5.1 功能定制

Enterprise 客户可申请定制功能开发：

- 需求评审：CSM 收集需求，2 周内提供评估
- 开发排期：按合同约定的开发资源安排
- 部署方式：私有云部署或专属环境
- 测试支持：提供 UAT 测试环境和专属测试资源
- 上线保障：新功能灰度发布，专属监控

### 5.2 集成服务

CloudDesk 提供 Enterprise 客户集成支持：

| 集成类型 | 说明 | 支持水平 |
|---------|------|---------|
| SSO 集成 | SAML 2.0 / OIDC | 全程协助 |
| CRM 集成 | Salesforce / HubSpot | 配置指导 |
| IM 集成 | Slack / Teams / 钉钉 | 配置指导 |
| 自定义 API | REST API 定制开发 | 全程协助 |
| 数据迁移 | 从现有系统迁移到 CloudDesk | 全程协助 |

### 5.3 培训服务

| 培训类型 | 时长 | 频率 | 费用 |
|---------|------|------|------|
| 管理员培训 | 2 小时 | 每季度 1 次 | 包含 |
| 终端用户培训 | 1 小时 | 每季度 1 次 | 包含 |
| 高级功能培训 | 1.5 小时 | 每半年 1 次 | 包含 |
| API 开发者培训 | 3 小时 | 每半年 1 次 | 包含 |
| 定制培训 | 自定义 | 按需 | 额外收费 |

## 6. 私有部署

### 6.1 私有部署选项

| 选项 | 说明 | 起订金额 |
|------|------|---------|
| 专属云环境 | 独立的 CloudDesk 实例部署在公有云上 | $50,000/年 |
| 私有云部署 | 部署在客户指定的私有云或数据中心 | $100,000/年 起 |
| 混合部署 | 核心数据在私有云，AI 服务在公有云 | $75,000/年 起 |

### 6.2 私有部署保障

- 数据不离开客户指定区域
- 客户可自行管理加密密钥（BYOK）
- 定期安全审计和渗透测试
- 独立升级窗口控制
- 专属运维团队支持

## 7. 定期回顾

### 7.1 月度运营回顾

- 频率：每月第 2 周
- 参与人：CSM + 客户关键联系人
- 内容：
  - 上月服务指标（可用性、响应时间、工单数）
  - 未解决问题跟进
  - 下月计划
  - 产品更新通知

### 7.2 季度业务回顾

- 频率：每季度末
- 参与人：CSM + 客户管理层 + CloudDesk 产品负责人
- 内容：
  - 季度 SLA 达成情况
  - ROI 分析和价值展示
  - 产品路线图对齐
  - 续约和增值服务讨论

### 7.3 年度战略回顾

- 频率：每年合同到期前 60 天
- 参与人：CSM + CloudDesk VP + 客户高管
- 内容：
  - 年度使用总结
  - 战略方向对齐
  - 续约谈判
  - 下年度规划

## 8. 账户管理

### 8.1 多团队管理

Enterprise 客户可创建多个团队，每个团队独立管理：

- 团队管理员可管理团队成员
- 团队间可配置数据共享策略
- 每个团队有独立的工单队列和报表
- CSM 负责整体账户管理

### 8.2 使用量报告

- 每周自动发送使用量报告
- 报告包含：AI 对话使用量、工单统计数据、SLA 达成率
- 自定义报告可通过 API 获取
- Enterprise 控制台提供实时使用量仪表盘

### 8.3 续约流程

1. 合同到期前 90 天：CSM 主动联系启动续约讨论
2. 合同到期前 60 天：年度战略回顾会议
3. 合同到期前 30 天：续约合同签署
4. 合同到期前 7 天：如未续约自动转为月付 Business 套餐
5. 合同到期后 30 天：如仍未续约，数据进入保留期
