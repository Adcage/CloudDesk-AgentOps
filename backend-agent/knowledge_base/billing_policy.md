# CloudDesk 账单政策

## 1. 政策概述

CloudDesk 账单政策规范了平台与客户之间的计费规则、账单处理流程和争议解决机制。本政策适用于所有 CloudDesk 订阅套餐，包括 Free、Business 和 Enterprise。

## 2. 计费周期

### 2.1 月度计费

- 月度套餐从订阅生效日期开始计费，每 30 天为一个计费周期
- 计费日在每月同日扣款（如 15 号订阅，次月 15 号扣款）
- 月末日期自动调整：1 月 31 日订阅，2 月按 28/29 日计费
- 月度套餐可在计费日前 3 天取消下月续费

### 2.2 年度计费

- 年度套餐一次性收取全年费用，享受 8.5 折优惠
- 年度计费周期为 365 天（闰年 366 天）
- 年度套餐不支持中途降级退款（退款政策另行规定）
- 年度到期前 30 天系统将发送续费提醒

### 2.3 计费通知

- 扣款前 7 天发送账单预告邮件
- 扣款成功后发送确认邮件
- 扣款失败后 24 小时内发送提醒
- 连续扣款失败 3 次后暂停服务并发送紧急通知

## 3. 重复扣费处理

### 3.1 重复扣费识别

系统通过以下规则自动识别重复扣费：

- 同一客户 ID、同一订阅计划、相同金额
- 扣款时间在 72 小时内出现两次或以上
- 订阅状态在扣款期间为 active

### 3.2 重复扣费处理流程

```
系统识别重复扣费 → 自动冻结重复扣款记录
→ 创建内部工单标记为 urgent
→ 通知客服团队人工确认
→ 确认后自动退款或转为账户余额
→ 通知客户处理结果
```

### 3.3 重复扣费预防措施

- 支付网关幂等性校验：同一订单号不允许重复扣款
- 订阅状态锁：扣款期间锁定订阅状态，防止并发请求
- 对账监控：每日自动对账，人工复核异常交易
- 客户自助：客户可在账单页面标记疑似重复扣费

### 3.4 重复扣费退款时限

- 自动识别并确认的重复扣费：1-2 个工作日内退款
- 客户手动申报的重复扣费：3-5 个工作日审核后退款
- 批量重复扣费事件：3 个工作日内统一处理

## 4. 账单争议

### 4.1 争议提交

客户如对账单有异议，可通过以下渠道提交争议：

- 客服智能体对话：说明争议原因和涉及金额
- 工单系统：提交 Billing Dispute 类别工单
- 邮件：billing-dispute@clouddesk.com

### 4.2 争议处理流程

```
客户提交争议 → 客服智能体分类（金额/类型）
→ 小额争议（< $50）：自动处理，3 个工作日内解决
→ 中额争议（$50-$500）：进入审核流程，5 个工作日内解决
→ 大额争议（> $500）：需经理审批，10 个工作日内解决
→ 客户收到处理结果通知
```

### 4.3 争议期间账单状态

- 争议金额部分的账单状态标记为 `disputed`
- 非争议部分正常计费
- 争议期间不会因争议金额触发逾期
- 争议期间服务不中断

### 4.4 争议升级

- 未在承诺时限内解决的争议自动升级
- Free 客户升级至 Senior Agent 处理
- Business 客户升级至 Billing Team Lead
- Enterprise 客户升级至专属客户经理

## 5. 发票说明

### 5.1 发票类型

| 发票类型 | 说明 | 适用场景 |
|---|---|---|
| 自动发票 | 系统自动生成，含详细订阅信息 | 所有付费订阅的每次扣款 |
| 增值税发票 | 含增值税信息的合规发票 | 需要增值税抵扣的企业客户 |
| 月度汇总发票 | 汇总当月所有交易 | Enterprise 客户 |
| 退款发票 | 记录退款详情 | 所有退款操作 |

### 5.2 发票内容

每张发票包含以下信息：

- 发票编号：格式 INV-YYYYMMDD-NNNN
- 客户名称和地址
- 订阅计划和计费周期
- 费用明细（基础费用、税费、折扣）
- 支付方式和交易 ID
- CloudDesk 公司注册信息和税号

### 5.3 发票获取

- 电子发票自动发送至注册邮箱
- 客户可在账户设置中配置发票接收偏好
- 支持在平台发票页面下载 PDF 版本
- Enterprise 客户可指定财务邮箱接收

### 5.4 发票修改

- 发票信息（名称、地址、税号）可在下次扣款前修改
- 已生成发票的修改需提交工单
- 修改发票信息不改变已生成发票的内容
- 增值税发票修改需要 3-5 个工作日

## 6. 付款方式

### 6.1 支持的付款方式

| 付款方式 | 处理时间 | 适用套餐 | 手续费 |
|---|---|---|---|
| 信用卡/借记卡 | 即时 | Free, Business | 无 |
| 银行转账（ACH） | 2-3 个工作日 | Business, Enterprise | 无 |
| 银行电汇 | 3-5 个工作日 | Enterprise | 银行收费 |
| PayPal | 即时 | Free, Business | 无 |
| 支票 | 7-10 个工作日 | Enterprise | 无 |

### 6.2 付款方式管理

- 客户可在账户设置中添加和管理付款方式
- 支持设置默认付款方式
- 支持配置备用付款方式（主方式失败时自动尝试）
- Enterprise 客户支持采购订单（PO）付款

### 6.3 付款安全

- 所有支付信息通过 PCI DSS Level 1 认证处理
- 不存储完整信用卡号，使用 token 化处理
- 支付网关使用 TLS 1.3 加密传输
- 异常支付行为自动触发安全审核

## 7. 逾期处理

### 7.1 逾期时间线

| 逾期天数 | 处理措施 |
|---|---|
| 第 1 天 | 发送逾期提醒邮件 |
| 第 3 天 | 第二次逾期通知，尝试自动扣款 |
| 第 7 天 | 服务降级通知（功能受限） |
| 第 14 天 | 服务暂停（仅保留数据，不可使用功能） |
| 第 30 天 | 数据保留期开始，服务取消 |
| 第 60 天 | 数据清除（不可恢复） |

### 7.2 逾期服务降级规则

- 逾期 7 天后：禁用高级功能，仅保留基础查看权限
- 逾期 14 天后：暂停所有功能，数据保留但不可访问
- 逾期期间产生的工单仍可查看，但不可创建新工单

### 7.3 逾期恢复

- 逾期期间付款成功后，服务在 1 小时内恢复
- 恢复后订阅状态自动从 suspended 变为 active
- 恢复后发送服务恢复确认邮件
- 累计逾期 3 次以上的客户需提前预付下期费用

## 8. 折扣与优惠

### 8.1 标准折扣

- 年度订阅：8.5 折（约节省 1.5 个月费用）
- 教育机构：7 折（需提供教育机构证明）
- 非营利组织：7.5 折（需提供注册证明）
- 新用户首月：5 折优惠（仅限 Business 套餐）

### 8.2 团购折扣

- 5-19 席位：9 折
- 20-49 席位：8.5 折
- 50 席位以上：联系销售团队获取定制价格

### 8.3 促销折扣

- 促销折扣有效期以活动公告为准
- 促销折扣不与标准折扣叠加
- 促销期间订阅的客户，续费按标准价格计费
- 已有订阅的客户参加促销活动，差价以账户余额形式返还

## 9. 税费处理

### 9.1 税费计算

- 税费根据客户账单地址自动计算
- 支持的税种：VAT、GST、Sales Tax、消费税
- 税率更新实时同步各国/地区税务数据

### 9.2 税费豁免

- 符合免税条件的企业可提交免税证明
- 免税审核通过后，后续账单自动免除对应税费
- 免税证明有效期 1 年，需每年更新
- 已缴税费如符合豁免条件，可申请退税

## 10. 账单记录与查询

### 10.1 账单历史

- 客户可在账户页面查看完整账单历史
- 支持按日期范围、金额、状态筛选
- 每笔交易记录包含：交易 ID、金额、状态、时间、支付方式

### 10.2 账单导出

- 支持 CSV 和 PDF 格式导出
- 支持自定义日期范围导出
- Enterprise 客户支持 API 批量导出
- 导出数据可用于企业财务系统对账

### 10.3 账单通知设置

- 扣款提醒：扣款前 3 天
- 账单生成：每月/每年计费日
- 付款确认：扣款成功后
- 逾期提醒：逾期后第 1、3、7 天
